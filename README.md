# Search Engine

A simple Python-based text search engine.

## Description
This project is a basic search engine written in Python. It searches through text documents and returns results based on a user-provided query. The project is intended for learning purposes and demonstrates file handling and text processing.

## How It Works
- Reads all `.txt` files from the `documents` folder
- Compares their contents against the search query
- Displays matching results in the terminal

## Requirements
- Python 3.x
- No external libraries required

## How to Run
1. Open a terminal in the project folder
2. Run the following command:
```bash
python find_correct_page.py
